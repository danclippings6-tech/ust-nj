# UST NJ — Business Playbook

## Brand

**Name:** UST NJ  
**Say it out loud:** “U-S-T N-J”  
**One sentence:** We are Class A/B operators for New Jersey underground storage tanks.  
**Never:** vague “compliance solutions” without saying Class A/B and UST.

### Why this name wins SEO + AI

| Weak brand | Strong brand |
|------------|--------------|
| TankSimple, EcoGuard, Apex Compliance | **UST NJ** |
| Requires explanation | Matches search: “UST operator NJ” |
| AI guesses | AI maps entity → Class A/B service |

## Product ladder

1. **Audit** — scorecard  
2. **Management** — our name on the Facility Certification Questionnaire (**the business**)  
3. **Problem response** — when DEP/equipment hits  

Honest line every time: *Owner keeps liability. We take the operator role.*

## Ideal customer

- Gas stations  
- Multi-site fuel groups in NJ  
- Fleet / card-lock / municipal  
- Generator UST facilities  
- New owners after a sale  

**Triggers:** manager quit, registration pending, buying a station, inspection coming, failed test, “I heard we need Class A/B.”

---

## 15-minute sales script

1. How many NJ sites? Fuel / fleet / generator?  
2. Who is Class A/B today?  
3. Any DEP letters or delivery holds?  
4. Explain UST NJ: licensed A/B on file + simple program; you still own liability  
5. Recommend audit-first or management  
6. Book next step same day  

**Close:**  
“You don’t need a 40-page proposal. You need a Class A/B name on the form and a program that doesn’t fall apart when the night clerk freezes at an alarm. That’s UST NJ.”
