How UST NJ works

No mystery. Six clear steps from first call to a boring, reliable UST program.

Free 15-minute call

Gas station, fleet, generator? One site or many? Who’s on the paperwork today? We tell you if audit-first or management-first is smarter.

No charge

UST compliance audit

We collect the basics. You get a short scorecard: what’s fine, what to fix, what’s urgent. Plain English only.

From $495 / site

Simple agreement

Readable scope: what UST NJ does as Class A/B, what you still do as owner, what problem response costs.

Clear scope

Our name goes on the UST program

We help designate UST NJ’s licensed Class A/B on your New Jersey Underground Storage Tank Facility Certification Questionnaire — the form NJDEP uses for A and B operators.

Core deliverable

Train Class C on site

Usually clerks/attendants. First response to alarms and spills. We train them so they’re not guessing. New hires can be trained as staff turns over.

Included in management

Run the boring program

Calendar. Records. Quick answers. Annual check-in. Abnormal stuff escalates to problem response so we don’t pretend everything is fine.

From $149 / mo
Owner reality check: Even when a contractor is designated Class A/B, New Jersey still holds the UST owner/operator responsible for compliance. UST NJ takes the operator role and program load — we don’t sell fake immunity.
